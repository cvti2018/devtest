# DevTest - Cadastro de EPI

Implemente os seguintes cadastros:

- Cadastro de empresa
- Cadastro de Função
- Cadastro de Funcionário
- Associar um ou vários EPIs a um funcionário

No cadastro de funcionários mostrar os EPIs associados a esse funcionário.

Requisitos para o desenvolvimento:
- Injeção de dependência
- TDD
- Persistência dos dado e regras de negócio do projeto Domain

---
## Informações gerais

1. Crie um branch e o utilize para enviar o código ao finalizar
2. O banco de dados já possui o cadastro de EPIs, não é necessário fazer o cadastro
3. Na tabela Funcionário EPI a data de troca e a data que um EPI já cadastrado foi substituído por outro do mesmo tipo
